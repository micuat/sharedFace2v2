sharedFace2v2
========

Face projection with KinectV2


Dependency
--------

* [HDFaceOsc](https://github.com/micuat/HDFaceOsc)
* [RoomAliveToolkit](https://github.com/Kinect/RoomAliveToolkit) for calibration
* openFrameworks 0.8.4 VS
* ofxOpenCv
* ofxOsc
* ofxXmlSettings
* ofxCv
